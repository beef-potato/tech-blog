# μTorrent继续下载

so 这是一个关于μTorrent软件在选择部分种子文件之后如何补充下载其他文件的文章。

![选中文件](https://cdn.jsdelivr.net/gh/beef-potato/images-hosting@master/blog/torrent.2rdb917d7ti0.webp)

以我的种子文件为例，我只下载了部分的教材，但是又想要其他的教材，在不移除原有种子的情况下，选中种子，下面有一个**文件**，可以看见各个文件的优先级，将其改为正常即可加入下载。

这之前我都是先移除种子，然后重新下载，让软件自己检验下载的文件，非常麻烦，这个功能还是方便的。
